Project Overview

The fundamentals of responsive coding for various screen sizes were explored in this demo exercise. Concepts including scalability and media queries were explored.

A simple design was provided. The goal of the demo was to teach how to replicate the design using HTML and CSS.

Understanding Grid

In order to understand how grids work in CSS, a visual was created to outline the organization of columns within the page.

Writing Code

Finally, code was written to replicate both the mobile and desktop layouts. As the user scales the site up and down, the site scales responsively.
