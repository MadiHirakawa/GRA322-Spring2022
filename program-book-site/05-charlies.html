Cafe Site Design & Code

Project Overview

Charlie's Cafe, located in ASU's College of Design North, is a staple among design students. A responsive single page site that outlines the menu, locations and other important information was sketched, designed and coded.

Research & Wireframes

After researching the content and layout of competitor cafe sites, wireframe sketches were constructed. The content of the site was taken from the existing cafe's menu and contact information. Details, including information hierarchy were considered in this process.

Digital Wireframes

Quick mockups of the pencil sketches were designed in Figma. Different versions for desktop, tablet and mobile were designed to ensure that the site is able to be viewed on various screen sizes.

Writing Code

The mockups were referenced as the project was brought into Visual Studio Code. First, HTML elements were written, then styling according to the mockup was applied with CSS classes.
